Picture.

Picture es una aplicación creada con React y basada en la API de pexels, con el fin de ofrecer una cantidad casi infinita de fotos y videos, esta página web no está desarrollada con fines comerciales, es simplemente una demostración del diseño de ui y la implementación de la API rest usando React, en una pagina web agradable a la vista e interactiva.
